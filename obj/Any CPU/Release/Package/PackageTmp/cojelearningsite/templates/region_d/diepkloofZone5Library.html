Diepkloof Zone 5 Library

Diepkloof Zone 5 Library is situated at ward 27 in Diepkloof Zone 5 Soweto. It is staffed by Senior Librarian, Senior Assistant Librarian, Library Assistant, general worker and a security officer. The library serves mostly the community of Diepkloof zone 2,3,4,5,6, Diepkloof extension, Ormonde, 3 high schools and a number of primary and pre-schools. The library provides story times to creches around, they come to the library every last Thursday of the month for an hour at 10h00. The library provides books every month to centres. The Library also provides home work assistance to primary schools, information Literacy, and services that train library users on how to find information in the library.
